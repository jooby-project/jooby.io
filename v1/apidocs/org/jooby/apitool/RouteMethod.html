Class RouteMethod

  • java.lang.Object
    • org.jooby.apitool.RouteMethod

  • public class RouteMethod
extends Object
    Contains all metadata for a Jooby route. It is similar to Route.Definition but with extra runtime information like parameters, return type and javadoc.
