props

Replace ${} expressions with application properties.

Make sure you’ve already set up the assets module in your project!

usage

assets {
 fileset {
   home: ...
 }
 pipeline {
   dev: [props]
   dist: [props]
 }
}

example

application.conf:

foo = bar

app.js:

(function (foo) {
  console.log(foo);
})("${foo}")

prints:

bar

options

delims

The delims options allow you to change the default delimiters ${ }:

assets {
 ...
 props {
   delims: [<%, %>]
 }
}

ignoreMissing

Unresolved properties results in NoSuchElement exception. Setting ignoreMissing skip missing properties:

assets {
 ...
 props {
   ignoreMissing: true
 }
}
