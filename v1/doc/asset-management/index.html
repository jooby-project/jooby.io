We do provide two modules for writing modern web applications:

  • assets: a custom framework for validate, transform, concatenate and minify assets. Useful, for web application with simple asset management.

  • frontend: install and run node locally. Allow you to use webpack, grunt, etc.. as well as modern UI libraries like [react.js], [vue.js] … useful for web application with complex asset management.
